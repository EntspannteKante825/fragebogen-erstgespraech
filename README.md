# Fragebogen Erstgespräch Unternehmenskunden

Professioneller Beratungsbogen für Versicherungsmakler im Erstgespräch mit Unternehmenskunden.

**Abgedeckte Bereiche:** bAV · bKV · Haftpflicht · Sach · Elektronik · Glas · Rechtsschutz · KFZ · Cyber · Personenversicherungen

**Features:**
- Modularer Aufbau (Sektionen ein-/ausklappbar)
- Autosave via localStorage
- Print-to-PDF optimiert
- Hochglanz Finance-Design (Navy + Gold)
- Gesprächsleitfaden-Fragen für jeden Themenblock

→ Einfach `index.html` im Browser öffnen oder via GitHub Pages bereitstellen.
